Doctoral Dissertation Proposal

A Practical Detection System for Government Student Loan Fund Agency

Developing a Rust/Python-Based Intrusion Detection System aligned with MITRE ATT&CK, MITRE D3FEND, NIST, and ISO 27001:2022

Executive Summary

This proposal outlines the development of a practical, hybrid Rust/Python-based intrusion detection and fraud prevention system for a government Student Loan Fund agency.

The research aligns perfectly with empirical and experimental methodologies, a two-year timeline, and the goal of building a functional tool with immediate practical applicability.

Key Objectives

  • Develop a hybrid Rust/Python intrusion detection system
  • Focus on proactive threat detection capabilities
  • Build a functional prototype for real-world deployment

Proposed Dissertation Topic

Dissertation Title

"Developing a Rust/Python-Based Intrusion Detection System for the Government Student Loan Fund Agency, Aligned with MITRE ATT&CK, MITRE D3FEND, NIST, and ISO 27001:2022"

Core Concept

The central concept is the design, development, and empirical validation of a novel intrusion detection system leveraging a hybrid technology stack. The system will utilize Rust for core, performance-critical components such as network packet capture, parsing, and real-time analysis, while employing Python for higher-level tasks including data aggregation, feature engineering, and machine learning implementation.

This hybrid approach combines Rust's memory safety and performance with Python's extensive data science ecosystem.

Technical Stack

Rust Performance Core
Python Analytics Layer

Primary Focus

Proactive threat detection within a government Student Loan Fund agency, focusing on identifying malicious activities in their early stages before significant damage occurs.

Alignment with Researcher Goals

Empirical & Experimental Methodologies

The research will be grounded in practical application and rigorous testing of a real-world system. The empirical component involves collecting and analyzing data from system operations, measuring key performance indicators such as detection accuracy, throughput, and resource utilization.

Experimental Design

  • • Simulated attacks based on MITRE ATT&CK framework
  • • Systematic evaluation of detection capabilities
  • • Statistical analysis of performance metrics

Practical Applicability Focus

Emphasis on immediate practical applicability over purely academic novelty. The primary goal is building a functional, robust tool that provides tangible value to security operations teams, rather than pursuing open-ended theoretical research.
