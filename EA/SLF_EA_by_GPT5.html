ชั้นที่ 2 — Data Architecture

Big Data Platform (Petabyte‑scale)

  • Object Storage — S3‑compatible, Erasure Coding (−40% space), Multi‑site Replication, Metadata Tags, Lifecycle → Cold Storage (−60% cost)
  • Distributed File System (HDFS) — 50+ nodes, HA NameNode, MapReduce/Spark, RW throughput ~ 10 GB/s, Replication Factor 3

Data Masking & PII Protection (PDPA)

  • Dynamic Masking — FPE/Tokenization, RBAC‑based reveal, Call Center เห็นเฉพาะ 4 หลักท้าย
  • Static Masking — Substitution/Shuffling/AES‑256/Nulling/Masking/Date Aging/Variance ฯลฯ สำหรับ Dev/Test
  • PII Discovery & Catalog — ML + Pattern, 30 ประเภท, Sensitivity 4 ระดับ, Heat Map & Compliance Dashboard

Data Exchange & API Standards

  • REST (OpenAPI 3.0), JSON Schema, OAuth2 + JWT, Rate‑limit (1,000 req/min/IP), Versioning, API Gateway + Dev Portal
  • Formats — XML (TH e‑GIF), JSON, CSV/Excel, HL7 FHIR, ISO 20022
  • Secure Sharing — TLS 1.3, SFTP/FTPS, Kafka Streaming (10+ หน่วยงาน), Digital Signature, Data Lineage + Audit Log

ชั้นที่ 3 — Application Architecture

Core Systems

  • DSL — ครอบคลุม end‑to‑end, ผู้ใช้งาน 7.15 ล้านบัญชี, 50,000 คำขอ/วัน, เปลี่ยนลำดับตัดชำระ/ปรับยอด/ปรับโครงสร้างหนี้ออนไลน์
  • CRM — 360° Profile, Omni‑channel, SLA‑based Ticketing, KB 5,000+ Q&A
  • Contact Center — Cloud IVR, Skill‑based Routing, 2,000 สาย/วัน, Screen Pop, Recording, Speech Analytics

Analytics Systems

  • BI Dashboards — Power BI (การเงิน/KPI 50+ หน้าปัด), Tableau (Exploratory), Near Real‑time 15 นาที, Drill‑down/Forecast/What‑if
  • Advanced Analytics — Credit Scoring, Churn/Default Prediction, Collection Optimization, Student Success, A/B Testing

Supporting Systems

  • Chatbot — NLP (ภาษาถิ่น/คำย่อ), 24/7, Coverage 80%, Accuracy ~92%, Sentiment → human handoff
  • SMS Gateway — 500k ข้อความ/วัน, OTP, Bulk/Personalized, Delivery Report, Two‑way SMS (~99.5% success)
  • DMS — 50M+ เอกสาร, OCR + AI Classification, Full‑text, Versioning, Digital Signature, Retention Policy
  • Legal & Institution Portals — สำหรับทนายความ/สถานศึกษา, API/Reporting

ชั้นที่ 4 — Technology Architecture

Infrastructure

  • HCI — รวม Compute/Storage/Network/Virtualization, ลดความซับซ้อน ~60%, DC Footprint −70%, Snapshot ทุก 4 ชม.
  • Oracle 19c EE — 10k TPS, RAC (HA), Data Guard (DR), Advanced Security
  • Data Center Network — 100 Gbps, SDN, Multi‑tenancy, LB, VXLAN

Cloud Services

  • LMS — 10k concurrent, 50TB + CDN, Progress/Assessment/Certificate
  • Backup 3‑2‑1 — Daily (02:00) Incremental, Space −80%, PITR 30 วัน, Archive 7 ปี
  • DR Site — ปทุมธานี, Active‑Passive, Sync Replication, RTO 4 ชม., RPO 15 นาที, DR Drill รายไตรมาส

End‑User Computing

  • Leasing Desktop 444 + Notebook 60 เครื่อง (36–48 เดือน)
  • Windows 11 Pro, M365, AV, On‑site 4 ชม., เครื่องสำรองหากซ่อม > 24 ชม.
  • ลด OPEX บำรุงรักษา ~35% เทียบการซื้อขาด

ชั้นที่ 5 — Security & Governance (ครอบคลุมทุกชั้น)

Framework

  • ISO/IEC 27001:2022 — ISMS ครบวงจร (องค์กร/บุคลากร/กายภาพ/เทคโนโลยี)
  • Zero Trust (ZTNA) — “Never Trust, Always Verify” ป้องกัน Lateral Movement

Protection

  • IPS/NGFW/NAC — L7, Zero‑Day/DDoS/SQLi/XSS, update ทุก 15 นาที
  • SOC/SIEM + AI — 24/7, Behavioral Baseline, −70% False Positive, Automated Playbook ~30 วินาที

Compliance & Access

  • PDPA (GPPC) — Consent, Data Subject Rights, Breach Notification
  • Audit Trail (Immutable) — Who/What/When/Where/Why เก็บ 3 ปี
  • RBAC/ABAC + PAM — Review สิทธิ์ทุก 3 เดือน, Auto‑Revoke เมื่อพ้นหน้าที่

หมายเหตุ: ปกป้องข้อมูลส่วนบุคคลผู้กู้กว่า 7.15 ล้านบัญชี; ความล้มเหลวอาจกระทบทางกฎหมาย/ชื่อเสียง/ความเชื่อมั่น จึงกำหนดมาตรการเชิงรุกทั้งเทคนิคและธรรมาภิบาลข้อมูล

KPI: Security

  • MTTD/MTTR ลดลง ไตรมาสต่อไตรมาส

KPI: Engagement

  • อัตราการใช้ Portal/Mobile/Chatbot เพิ่มขึ้น
  • เวลาบริการเฉลี่ย (SLA) ลดลง

  • Availability ระบบหลัก ≥ 99.9%
