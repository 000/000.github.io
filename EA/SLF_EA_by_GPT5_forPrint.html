ภาพรวมสถาปัตยกรรม (TOGAF EA Stack)

Security & Governance (ครอบคลุมทุกชั้น)

  • Zero Trust (ZTNA) — “Never Trust, Always Verify”, ป้องกัน Lateral Movement
  • ISO/IEC 27001:2022 — ISMS ครบ 4 มิติ (องค์กร/บุคลากร/กายภาพ/เทคโนโลยี)
  • SOC/SIEM + AI — ลด False Positive ~70%, Automated Playbook ~30 วินาที
  • PDPA (GPPC), Audit Trail (Immutable, เก็บ 3 ปี), RBAC/ABAC + PAM

Business Architecture

  • Customer Journey (End‑to‑End): ยื่นกู้ → อนุมัติ → เบิกจ่าย → ชำระหนี้ → ติดตามหนี้/คดี
  • Stakeholder Engagement: ผู้กู้ยืม, สถานศึกษา (4,000+), สำนักงานทนายความ (200+), หน่วยงานเชื่อมโยง (50+)
  • ผลลัพธ์: เวลาบริการสั้นลง, ประสบการณ์ดิจิทัลดีขึ้น, รายงาน/แดชบอร์ดเพื่อการตัดสินใจ

Data Architecture

  • Big Data (Petabyte‑scale): Object Storage (S3‑compatible, Erasure Coding, Lifecycle→Cold), Distributed FS (HDFS, 50+ nodes, HA, Spark/MapReduce)
  • PII Protection (PDPA): Dynamic/Static Data Masking (FPE/Tokenization/AES‑256), Discovery & Catalog, Sensitivity 4 ระดับ
  • Data Exchange & API Standards: OpenAPI 3.0, OAuth2+JWT, TH e‑GIF XML/JSON/CSV, TLS1.3, SFTP/FTPS, Kafka Streaming

Application Architecture

  • DSL (Digital Student Loan): ครอบคลุมสมัคร‑อนุมัติ‑เบิกจ่าย‑ชำระ‑ปรับโครงสร้างหนี้ (ตาม พ.ร.บ. ฉบับที่ 2), รองรับ 7.15M บัญชี ~50k คำขอ/วัน
  • CRM & Contact Center: Omni‑channel, SLA Ticketing, Cloud IVR/Skill Routing, Speech Analytics
  • Analytics: Power BI/Tableau (Near Real‑time ~15 นาที), Advanced ML (Credit/Default/Churn/Collection Optimization)
  • Supporting: Chatbot (NLP TH, 24/7, Accuracy ~92%), SMS Gateway (500k/วัน), DMS (50M+ ฉบับ, OCR/AI Classify)

Technology Architecture

  • Infrastructure: HCI (ลดความซับซ้อน ~60%, Snapshot/4ชม.), Oracle 19c EE (RAC/HA, Data Guard/DR), DC Network 100Gbps (SDN/VXLAN)
  • Cloud Services: LMS (10k concurrent, 50TB + CDN), Backup 3‑2‑1 (Daily 02:00, PITR 30 วัน, Archive 7 ปี)
  • DR Site: ปทุมธานี, Active‑Passive, Sync Replication, RTO 4 ชม., RPO 15 นาที (ทดสอบรายไตรมาส)
  • End‑User Computing: เช่า Desktop 444 + Notebook 60, Windows 11/M365/AV, On‑site 4ชม., ลด OPEX บำรุงรักษา ~35%

Customer Journey แบบ End‑to‑End (แนวตั้งเพื่อการพิมพ์)

1) ยื่นกู้ (Application)

  • DOPA Digital ID, e‑Studentloan Portal/Mobile
  • AI Assistant + OCR + Rule Engine ตรวจคุณสมบัติทันที
  • Save Draft, Progress Bar, นัดหมายออนไลน์
ระยะเวลาเป้าหมาย: ≈ 15 นาที

2) อนุมัติ (Approval)

  • Workflow อัตโนมัติ + AI Scoring
  • ยืนยันเอกสารผ่าน API กับ 5 หน่วยงานภาครัฐแบบ Real‑time
  • e‑Meeting + Digital Signature, แจ้งผล SMS/Email/Push
ระยะเวลาเป้าหมาย: 14 → 7 วัน

3) เบิกจ่าย (Disbursement)

  • PromptPay/บัญชีธนาคาร, แบ่งงวดตามภาคเรียน
  • Check‑in กับสถานศึกษาก่อนโอน, e‑Receipt, Refund อัตโนมัติ
  • Smart Contract เงื่อนไขการจ่าย, Payment Tracking
ระยะเวลาเป้าหมาย: ≤ 3 วันทำการ

4) ชำระหนี้ (Repayment)

  • 10+ ช่องทาง (Mobile/ATM/QR/Direct Debit/Credit/ฯลฯ)
  • คำนวณดอกเบี้ยรายวัน, Flexible Payment, Auto‑reminder 7/3/1 วัน
  • Installment Calculator, เอกสารลดหย่อนภาษี

5) ติดตามหนี้ & คดี (Collection & Legal)

  • Collection Scoring 5 ระดับ + Predictive Analytics (คาดการณ์ 30 วัน)
  • Omni‑channel (SMS/Call/Email/LINE) + Chatbot เจรจาเบื้องต้น
  • Field Routing, Legal Portal/Court Integration, ปรับโครงสร้างหนี้ออนไลน์

KPI: Security

  • MTTD/MTTR ลดลง รายไตรมาส
  • เหตุการณ์ High/Critical ลดลงต่อเนื่อง
  • เหตุเข้าถึง PII โดยมิชอบ = 0

KPI: Engagement/Service

  • อัตราใช้ Portal/Mobile/Chatbot เพิ่มขึ้น
  • SLA เวลาบริการเฉลี่ยลดลง
  • CSAT/NPS ดีขึ้น

KPI: Operations

  • Availability ระบบหลัก ≥ 99.9%
  • เวลารอบงาน (ยื่นกู้/ปรับโครงสร้าง/ติดตามหนี้) สั้นลง
  • ต้นทุนต่อธุรกรรมลดลง
