Designing the Next-Generation AI Accelerator System

A comprehensive analysis of Intel's Gaudi 3 architecture for high-density AI workloads, delivering 4TB of HBM2E memory across 32 accelerators with PCIe Gen 5 connectivity.

Executive Summary

An Intel-based AI accelerator system meeting all specified requirements can be designed using Intel Gaudi 3 accelerators. A logical node would comprise four Universal Baseboards (UBBs), each housing eight Gaudi 3 OAM modules (HL-325L), totaling 32 accelerators.

Proposed Solution: Intel Gaudi 3 Accelerators

Gaudi 3 Architecture Overview

The Intel Gaudi 3 AI accelerator is built on a 5nm process technology and represents a significant leap in performance and features compared to the previous Gaudi 2 generation [203]. It is designed to deliver high throughput for both training and inference of large-scale AI models, including Large Language Models (LLMs).

Key Performance Metrics

  • FP8 Compute: 1,835 teraflops per accelerator [203]
  • BF16 Support: 4× more AI compute vs. Gaudi 2 [203]
  • HBM2E Memory: 128GB per card [203]
  • Memory Bandwidth: 3.7TBps (1.5× Gaudi 2) [203]

Advanced Features

  • Integrated Networking: 24 × 200Gbps Ethernet ports [74]
  • Host Interface: PCIe Gen 5.0 x16 [74]
  • RoCE v2 Support: RDMA over Converged Ethernet
  • Form Factor: OAM modules for high density

Alternative: Intel Data Center GPU Max Series

Ponte Vecchio Overview

The Intel Data Center GPU Max Series (Ponte Vecchio) is a general-purpose GPU architecture designed for high-performance computing (HPC) and AI workloads. It features a complex chiplet-based design with various tile types manufactured on different process nodes.

Key Capabilities

  • • High memory bandwidth (HBM2e)
  • • Significant FP32/FP64 compute performance
  • • AI-optimized data types (BF16, FP16)
  • • Xe Matrix Extensions (XMX)
  • • Used in Aurora supercomputer

Target Workloads

  • • Traditional HPC simulations
  • • Scientific computing
  • • General AI workloads
  • • Mixed HPC/AI applications

Ponte Vecchio might be more appropriate for mixed HPC/AI workloads or where its specific HPC capabilities are paramount, but Gaudi 3 better meets the stated requirements.
